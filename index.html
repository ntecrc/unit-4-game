Infinity Stones Collector!!

Game instructions:


A random number will be shown at the start of the game.

The gems at the bottom each have random values, with a click of any gem you will get a random number thats added to your score.

You can only win if your score matches the starting random number, exceed the computers number and you lose the game

Everytime the game restarts the crystals number changes and so does the random number given at the beginning
